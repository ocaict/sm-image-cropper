# Future Features Checklist

This document outlines new features planned for future development of the Social Image Resizer/Cropper.

## Core Editing Enhancements

- [ ] **Image Adjustments**
    - [ ] Brightness slider
    - [ ] Contrast slider
    - [ ] Saturation slider
    - [ ] Hue/Temperature adjustment
- [ ] **Filters & Presets**
    - [ ] Basic filters (Grayscale, Sepia, Vintage)
    - [ ] "Instagram-style" preset collection
- [ ] **Watermarking & Branding**
    - [ ] Upload logo (PNG/SVG) support
    - [ ] Drag-and-drop logo positioning
    - [ ] Opacity control for watermark
    - [ ] Text-based watermark generator

## Application Features

- [ ] **PWA (Progressive Web App) Support**
    - [ ] Installable on Desktop/Mobile
    - [ ] Offline capabilities (Service Worker caching)
    - [ ] App manifest configuration
- [ ] **Direct Social Sharing**
    - [ ] Implement Web Share API (native share sheet)
    - [ ] Direct links to Twitter/Facebook composer
- [ ] **Keyboard Shortcuts**
    - [ ] Zoom In/Out (`+` / `-` or `Ctrl` + Scroll)
    - [ ] Pan (Space + Drag)
    - [ ] Next/Prev Image (Arrow keys)
    - [ ] Delete Item (`Delete` / `Backspace`)

## Creative Tools

- [ ] **Text Overlay**
    - [ ] Add text layer
    - [ ] Font selection (Google Fonts integration)
    - [ ] Color, size, and weight controls
- [ ] **Templates**
    - [ ] Pre-defined frames (e.g., Polaroid, Film strip)
    - [ ] Create custom user templates

## Performance & UX

- [ ] **Drag & Drop Reordering**
    - [ ] Reorder batch list by dragging
- [ ] **History / Undo Redo**
    - [ ] Global undo/redo stack for all actions (crop, filter, etc.)
